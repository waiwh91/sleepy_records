# Sleepy Records

A handmade, single-page website for Sleepy Records. It is plain HTML, CSS, and
JavaScript, so it can be published directly with GitHub Pages.

## Publish with GitHub Pages

1. Push these files to a GitHub repository.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select the main branch and `/ (root)`.

## Add photography

Create `assets/photos/`, add web-sized JPG or WebP files, then replace the image
paths in `index.html`. Images around 1800px wide and under 1 MB work well.
